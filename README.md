# Penguin MLP

Implementation of a multilayer perceptron (MLP) from scratch using NumPy.

## Features

- Forward propagation
- Backpropagation
- SGD optimizer
- Numerical gradient checking
- Classification and regression tasks
- Comparison with PyTorch implementation

## Dataset

Palmer Penguins dataset:
https://archive.ics.uci.edu/dataset/690/palmer+penguins-3

## Project Structure

```text
src/            # source code
notebooks/      # experiments
reports/        # results and notes
```

## Installation

```bash
pip install -r requirements.txt
```
